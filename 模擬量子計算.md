1. 核心技術整合：從「基礎指標」提升至「科學預測」
該 Repo 包含了一些高效能的科學計算與預測工具，可以直接強化你的 StrategySelector：

TimesFM (Google's Time Series Foundation Model):
整合價值：你目前的策略主要基於 RSI、MACD 等傳統指標。整合 timesfm 技能後，你可以提供一個「AI 預測路徑」功能，利用 Google 的時間序列大模型對個股未來 5-10 個 tick 的走勢進行機率性預測。
實作位置：放入 server/python/，作為一個新的微服務端點。
Parallel-Web (多線程網頁研究):
整合價值：你的計畫中提到了 global_sentiment（全球情緒分）。使用 parallel-web 技能，AI Agent 可以同時並行掃描多個財經新聞源（如 CNBC, Bloomberg, 鉅亨網），並在幾秒鐘內彙整出目前市場對特定標的（如 2330.TW）的綜合情緒。
Polars / Dask (高效能資料處理):
整合價值：對於回測（Backtest）功能，如果處理多年的 Tick 資料，Node.js 可能會遇到瓶頸。借鑑該 Repo 的 polars 技能，在 Python 端使用 Polars 進行並行化回測，速度會比一般 Pandas 快 10 倍以上。
